# Multi-Class Object Classification using Machine Learning

## Dataset
Caltech-101 Dataset

## Problem Statement
Design a multi-class object classification system using Machine Learning with feature engineering and ensemble learning.

## Methodology

Pipeline:

Image → Resize → HOG Feature Extraction → Random Forest → Evaluation

## Feature Engineering
Histogram of Oriented Gradients (HOG)

## Machine Learning Algorithm
Random Forest Classifier (Ensemble Learning)

## Evaluation Metrics

- Accuracy
- Confusion Matrix
- Cross Validation

## Results

Accuracy achieved: 72.38%

## Tools Used

- Python
- Google Colab
- OpenCV
- Scikit-learn
